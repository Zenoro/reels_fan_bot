# Скачиватель рилсов для тг

## Установка

1. Склонировать этот репозиторий
2. Подготовить виртуальное окружение и библиотеки с помощью скрипта:
    ```
    . setup.sh
    ```

## Запуск

```
python3 src/bot.py
```

## Использование

1. Добавить бота в чат и дать ему права админа
2. Прислать ссылку на рилс в чат
3. Кайфовать
